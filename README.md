# Portfolio-Website

## Admin Dashboard

The admin dashboard lets you manage projects, certificates, testimonials, and blog posts from `admin.html`.

### Setup

1. Install backend dependencies:
   - `cd server`
   - `npm install`
2. Create an admin account:
   - `npm run create-admin -- --email you@example.com --password yourPassword`
3. Start the server:
   - `npm start`

The site and API will be available at `http://localhost:3000`, and the admin dashboard at `http://localhost:3000/admin`.
